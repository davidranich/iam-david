homePi

A smart home hub to assist with day-to-day tasks and life, provide security in your living space, multimedia storage and streaming, and more. The goal is to replace all of my other smart devices along with releasing the code for those seeking an alternative. Check back for updates and releases.

Technologies used for this project (so far): JavaScript, PHP, MySQL (Maria DB), Batch, and Shell.



The following systems will be included with homePi:

  • Automatic Door unlock/lock w/ RFID
  • Security System w/ camera and motion sensor outside door
  • Plex Server
  • File Server
  • Voice Commands
  • Spending Ledger (Debit & Credit)
  • Manage Planned Tasks
